# Carpeta de evidencias

Guarda aqui tus capturas de pantalla.

Evidencias minimas recomendadas:

1. Consola mostrando un error inicial.
2. Codigo corregido en VS Code.
3. Sistema funcionando en el navegador.
4. Tabla con registros validados.
5. Bitacora de depuracion diligenciada.
6. Mapa conceptual o notes, si lo hiciste como imagen.

Puedes usar nombres como:

- evidencia-01-error-consola.png
- evidencia-02-codigo-corregido.png
- evidencia-03-sistema-funcionando.png
- evidencia-04-tabla-resultados.png
