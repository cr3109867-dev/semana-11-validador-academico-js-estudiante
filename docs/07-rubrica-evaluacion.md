# 07 - Rubrica de evaluacion

Puntaje sugerido: 100 puntos.

| Criterio | Descripcion | Puntos |
|---|---|---:|
| Identificacion de errores | Reconoce errores de sintaxis, referencia, tipo, logica y eventos. | 20 |
| Correccion tecnica | Corrige el codigo sin romper la estructura general del sistema. | 25 |
| Depuracion sistematica | Usa consola, pruebas, lectura de mensajes y correcciones paso a paso. | 20 |
| Control de flujos | Aplica correctamente condicionales, comparaciones, `switch` y ciclos. | 15 |
| Documentacion | Completa bitacora, cuestionario, notes/mapa conceptual y evidencias. | 15 |
| Presentacion final | Entrega ordenada, comprimida y con evidencias verificables. | 5 |

## Niveles de desempeno

### Superior

El sistema funciona, no presenta errores criticos en consola, la bitacora es clara, las respuestas muestran comprension y las evidencias son completas.

### Alto

El sistema funciona en lo principal, hay pequenos detalles por mejorar, la bitacora esta diligenciada y las pruebas principales fueron realizadas.

### Basico

El sistema funciona parcialmente, algunos errores persisten o la documentacion esta incompleta.

### Bajo

El sistema no funciona, no hay evidencia suficiente de depuracion o la entrega esta incompleta.
