# 09 - Reflexion final

Responde con honestidad tecnica.

## 1. Que aprendiste esta semana?
Que los errores en JavaScript pueden ser de sintaxis, referencia, tipo o lógica, y que la consola es clave para depurarlos.


## 2. Cual fue el error mas importante que corregiste?
El de asignación estado = "No aprobado", porque no generaba error en consola pero alteraba la lógica del sistema.



## 3. Como usaste la consola del navegador?
Para leer mensajes de error, ubicar la línea afectada y verificar valores con console.log().

## 4. Que estructura de control de flujo comprendiste mejor?
El if / else if / else, porque me permitió clasificar correctamente los estudiantes


## 5. Como aplicarias este aprendizaje en un proyecto real de tu comunidad o institucion?
En sistemas de registro académico o comunitario, validando datos y mostrando mensajes claros.


## 6. Que mejorarias si tuvieras mas tiempo?
La interfaz de usuario, validaciones más completas y documentación con capturas de pantalla.
