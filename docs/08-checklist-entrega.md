# 08 - Checklist de entrega

Antes de subir tu ZIP, verifica:

- [ ] El proyecto abre correctamente en VS Code.
- [ ] `index.html` se ejecuta en el navegador.
- [ ] La consola no muestra errores criticos.
- [ ] El formulario valida campos vacios.
- [ ] El formulario valida correo.
- [ ] El formulario valida edad.
- [ ] El formulario valida nota.
- [ ] El formulario valida asistencia.
- [ ] Los registros validos se agregan a la tabla.
- [ ] El estado academico se calcula correctamente.
- [ ] El boton limpiar funciona.
- [ ] El boton cargar casos de prueba funciona.
- [ ] El resumen se actualiza.
- [ ] Complete la bitacora de depuracion.
- [ ] Respondi el cuestionario.
- [ ] Complete el mapa conceptual o notes.
- [ ] Agregue capturas en la carpeta `evidencias/`.
- [ ] Comprimi la carpeta completa en ZIP.
