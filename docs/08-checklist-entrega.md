# 08 - Checklist de entrega

Antes de subir tu ZIP, verifica:

[x] El proyecto abre correctamente en VS Code.

[x] index.html se ejecuta en el navegador.

[x] La consola no muestra errores críticos.

[x] El formulario valida campos vacíos.

[x] El formulario valida correo.

[x] El formulario valida edad.

[x] El formulario valida nota.

[x] El formulario valida asistencia.

[x] Los registros válidos se agregan a la tabla.

[x] El estado académico se calcula correctamente.

[x] El botón limpiar funciona.

[x] El botón cargar casos de prueba funciona.

[x] El resumen se actualiza.

[x] Completé la bitácora de depuración.

[x] Respondí el cuestionario.

[x] Completé el mapa conceptual o notas.

[x] Agregué capturas en la carpeta evidencias/.

[x] Comprimí la carpeta completa en ZIP.
