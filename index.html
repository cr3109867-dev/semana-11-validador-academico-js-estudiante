Validador académico comunitario

Sistema evaluativo para practicar errores frecuentes en JavaScript, depuración sistemática y control de flujos desde Visual Studio Code.

Reto principal

Registro y validación de estudiantes

El sistema debe validar los datos de un estudiante, clasificar su estado académico y registrar el resultado en la tabla. El archivo JavaScript contiene errores intencionales que debes encontrar, corregir y documentar.

Ruta de depuración

  1. Abre el proyecto en VS Code.
  2. Ejecuta index.html con Live Server o en el navegador.
  3. Abre la consola con F12 o Ctrl + Shift + I.
  4. Lee el primer error y ubica el archivo/línea.
  5. Corrige una cosa a la vez.
  6. Prueba de nuevo y completa la bitácora.

Preguntas guía para validar el sistema

1. Sintaxis

¿El navegador puede leer todo el archivo JavaScript sin detenerse?

2. Referencias

¿Los identificadores usados en JavaScript coinciden con los ID del HTML?

3. Tipos de datos

¿Los valores numéricos se convierten antes de comparar o calcular?

4. Flujo lógico

¿Las condiciones clasifican correctamente al estudiante?
